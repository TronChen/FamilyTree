框架與工具
---
* 設計模式：MVVM、Singleton
* 實作功能：RecyclerView (家族樹產生、聊天系統)、登入 (Firebase Authorization)
* Jetpack：ViewModel、LiveData、Lifecycle、Data Binding、Navigation
* 第三方工具：ZXing、MaterialCalendarView、QRGenerator、ImagePicker
* 分析工具：Firebase Crashlytics

開發環境
---
* Android Studio 4.0+
* Android SDK 26+
* Gradle 4.0.2+
